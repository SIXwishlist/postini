h1. postini4r


h2. What

postini4r is a Ruby wrapper library for the "Postini":http://www.postini.com/ SOAP API (Early Access Program).

postini4r aims to be fully compliant with the Postini API, starting off with what I need first, and then expanding to eventually cover all the available commands in the Batch Command Reference.

h2. Requirements

* postini4r requires that you gain access to the Postini Early Access Program and an API key. You need to contact your Postini Service Provider (or Postini directly if you deal with them).
* soap4r

h2. Installing

<pre syntax="ruby">sudo gem install postini4r</pre>
